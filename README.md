# Dodge the Creepers

This project is a simple 2D game built with Godot following the official tutorial at the Godot documentation: https://docs.godotengine.org/en/stable/getting_started/first_2d_game/

## What the game is

You control a player character and must avoid incoming enemies while surviving as long as possible. The game includes:

- player movement and collision handling
- enemy spawning and movement
- score tracking
- a game over screen and restart flow

## Project structure

- `main.tscn` and `main.gd` — game entry point and main gameplay logic
- `player.tscn` and `player.gd` — player character scene and behavior
- `mob.tscn` and `mob.gd` — enemy mob scene and behavior
- `hud.tscn` and `hud.gd` — score and game state UI

## How to run

1. Open the project folder in Godot.
2. Open `main.tscn` or the project root.
3. Press Run to start the game.

## Notes

This repository is a beginner-friendly example of a Godot 2D game and is intended for learning and experimentation.
